# UPyun Upload & Delete

前端 Vue.js + 后端 Node.js 实现又拍云（https://www.upyun.com/） 上传、删除文件。

---

## Usage

### Install

```
npm install
```

### Run

```
npm start
```
> http://localhost:8082/upload.html


## UPyun

### View uploaded files

```
curl -u <operator>:<password> http://v0.api.upyun.com/<bucket>/
```
